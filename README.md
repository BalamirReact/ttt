# Tic-Tac-Toe

AI for Tic-Tac-Toe in Node.js

## Commands

```bash
# Install dependencies
$ npm install

# build es6 source files in `/src` to `/lib`
$ npm run build

# watch and build es6 source files in `/src` to `/lib`
$ npm run watch 

# run compiled files, etc.
$ node lib/testRun.js

# run cli game
$ npm run game

# tests
$ npm test 
```
